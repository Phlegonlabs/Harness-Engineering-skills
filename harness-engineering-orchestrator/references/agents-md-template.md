# AGENTS.md / CLAUDE.md Template Guide

The root `AGENTS.md` and `CLAUDE.md` are now thin entry files; the full specification is delegated to `docs/ai/`.

1. [01 Entry File](./agents-md-template/01-entry-file.md)

## Rules

- `AGENTS.md` and `CLAUDE.md` must have identical content
- The root files must retain:
  - Read confirmation
  - Compatibility-entry guidance for humans / external tools
  - Dependency direction summary
  - Prohibited operations summary
  - Task Checklist summary
- Detailed rules are in `docs/ai/`
