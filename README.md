# Harness Engineering Skills

Public skill repository for Harness Engineering workflows.

This repository currently publishes `harness-engineering-orchestrator`, an orchestration skill for turning a new or existing software project into a repo-backed delivery loop from discovery through validated execution.

## Language

- English: [README.en.md](README.en.md)
- 中文: [README.zh-CN.md](README.zh-CN.md)

## Install

```bash
npx skills add https://github.com/Phlegonlabs/Harness-engineering-skills --skill harness-engineering-orchestrator
```
